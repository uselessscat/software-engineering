1. Origen  
    - Google, 2011  
    - Contexto: complejidad creciente de la web y límites de JavaScript  
2. Filosofía y principios  
    - Inicialmente pensado como mejora/reemplazo de JS  
    - Navegación con Dart VM embebida (Dartium)  
3. Evolución  
    - Dart 1.x: tipado opcional, enfoque web  
    - Abandono del VM en el navegador, enfoque en compilación a JavaScript  
    - Aparición de Flutter y reorientación total del lenguaje  
    - Dart 2: type system fuerte, sintaxis renovada, APIs limpias  
    - Dart 3.x: sound null safety, records, pattern matching, sealed classes  
4. Hitos y releases importantes  
    - Null safety (2.12)  
    - Records, patterns, sealed (3.x)  
    - Roadmap: multiplataforma, WASM, server-first, runtimes nativos
