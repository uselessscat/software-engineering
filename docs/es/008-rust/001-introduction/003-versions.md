1. Versionado y ciclos de vida  
    - Canales: stable / beta / nightly  
    - SemVer en crates  
2. Compatibilidad entre versiones  
    - Promesa de estabilidad  
    - Librería estándar estable (`std`)  
3. Diferencias entre runtimes  
    - Edition 2015 / 2018 / 2021 / 2024  
    - Cambios de sintaxis, módulos, async/await  
4. Herramientas para manejar versiones  
    - `rustup`  
    - Toolchains por proyecto (`rust-toolchain.toml`)  
    - `cargo fix --edition`  
    - Lints para migración
