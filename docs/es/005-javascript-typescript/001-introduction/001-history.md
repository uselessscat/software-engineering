1. Origen  
    - JavaScript en el navegador (Brendan Eich, 1995)  
    - Surgimiento de TypeScript como superconjunto tipado  
2. filosofia y principios  
    - Diseño de TypeScript: tipado opcional, verificación en compilación  
    - “Any valid JavaScript is valid TypeScript”  
    - Influencias de C#, Java y lenguajes estáticos  
3. Evolución  
    - Evolución ECMAScript: ES3, ES5, ES2015+  
    - Transición JS: funciones → clases/módulos/async-await  
    - Expansión de runtimes: Node.js, Deno, Bun  
    - Evolución del tooling: bundlers, linters, transpiladores  
4. Hitos y releases importantes  
    - Impacto de Node.js en uso fuera del navegador  
    - Adopción y uso de TypeScript en monorepos y grandes bases de código  
    - Crecimiento del ecosistema moderno (Vite, esbuild, SWC)
