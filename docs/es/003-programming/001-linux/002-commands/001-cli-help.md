# Ayuda y documentación

## Herramientas de manual

* `man`: secciones, navegación, búsqueda
* `info`: documentación GNU avanzada

## Búsqueda de comandos

* `--help` y `-h`
* `whatis` y `apropos`

## Historial y autocompletado

* Navegación del historial (`↑`, `↓`, `Ctrl+R`)  
* `history`, ejecución por número (`!42`)  
* Expansiones del historial (`!!`, `!$`, `!*`)  
* Persistencia y configuración del historial
