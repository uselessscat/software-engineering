# Transformación de texto

## Herramientas básicas de flujo

* `cut`: dividir por columnas o delimitadores
* `sort`: orden alfabético, numérico, reverso
* `uniq`: filtrado y conteo

## Transformación avanzada

* `tr`: reemplazo o eliminación de caracteres
* `sed`: sustitución, edición en línea, filtros
* `awk`: procesamiento por campos, cálculos y reportes
