# Búsqueda de archivos y contenido

## Búsqueda de archivos

* `find`: filtros por nombre, tamaño, permisos, fechas
* Acciones con `-exec`, `-delete`, `-print0`
* `locate`: búsqueda indexada, uso con `updatedb`

## Búsqueda dentro de archivos

* `grep`: regex básicas, flags comunes (`-i`, `-r`, `-E`, `-v`)
* Patrones típicos y uso en pipelines
