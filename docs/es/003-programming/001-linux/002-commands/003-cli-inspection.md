# Inspección y lectura de archivos

## Lectura directa

* `cat`: concatenar y mostrar
* `head` / `tail`: primeras y últimas líneas
* `tail -f`: seguimiento en tiempo real

## Visualización paginada

* `less`: navegación interactiva, búsqueda, saltos
* Flags útiles (`-S`, `-N`)

## Métricas rápidas

* `wc`: conteo de líneas, palabras y bytes
