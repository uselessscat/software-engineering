# Navegación del sistema de archivos

## Comandos básicos

* `ls`: listado, flags `-l`, `-a`, `-h`
* `cd`: cambio de directorio, `cd -`, rutas relativas/absolutas
* `pwd`: ruta actual física vs lógica
* `tree`: visualización jerárquica

## Navegación eficiente

* `pushd` y `popd`
* Atajos (`~`, `.`, `..`)
* Variables de entorno (`CDPATH`)
