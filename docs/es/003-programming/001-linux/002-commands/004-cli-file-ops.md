# Manipulación básica de archivos y directorios

## Operaciones principales

* `cp`: copia recursiva, preservación de atributos
* `mv`: mover o renombrar
* `rm`: borrado seguro, uso cuidadoso de `-r` y `-f`
* Copias atómicas y patrones de reemplazo seguro

## Creación y mantenimiento

* `mkdir`: creación recursiva (`-p`)
* `touch`: crear archivos vacíos, actualizar timestamps
