# Globbing y expansiones del shell

## Comodines

* `*` para coincidencias amplias
* `?` para un carácter
* `{a,b}` para listas de patrones

## Expansiones

* Variables (`$VAR`)
* Sustitución de comandos (`$( )`)
* Expansión aritmética (`$(( ))`)
* Expansión de variables, quotes y escaping  
* Orden de evaluaciones del shell
